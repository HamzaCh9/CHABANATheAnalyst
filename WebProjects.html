Movies Search APP

with TMDB Api (themoviedb Database)

In this project, I developed a Movie Search App using HTML, CSS, JavaScript, and the TMDB API (The Movie Database). The app allows users to search for movies and retrieve detailed information about them.

The app's interface is visually appealing and user-friendly. Upon landing on the home page, users are presented with a randomly generated list of movies for an element of discovery.

To search for a specific movie, users can enter its name into the search bar. The app communicates with the TMDB API to fetch a list of movies with matching names. Results are displayed neatly, showing details like title, release date, rating, and a brief overview.

The TMDB API integration provides comprehensive movie information, including cast, genres, and production companies. This empowers users to make informed decisions about what movies to watch or explore further.

By developing this Movie Search App, I showcased my skills in front-end web development using HTML, CSS, and JavaScript. I also demonstrated my ability to work with external APIs, as I integrated the TMDB API to fetch real-time movie data.

This project represents my proficiency in web development and my passion for movies. I'm excited to include the Movie Search App in my portfolio as an example of a functional and visually appealing application that enhances the movie-watching experience.
